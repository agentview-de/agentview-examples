# Hello World

A polished starter display for AgentView. It uses one fullscreen HTML file with a small amount of vanilla JavaScript for the live clock and date.

## Preview

Open `index.html` in your browser.

## Send to AgentView

Use the agentView dashboard:

1. Open the dashboard at https://agentview.de.
2. Open your display.
3. Click `Send`.
4. Switch to the `HTML` tab.
5. Paste the contents of `index.html`.
6. Click `Send HTML now`.

You can also connect an AI agent through MCP at https://agentview.de/mcp and ask it to send this example to one of your displays.

## Customize

Edit `index.html` and change the headline, message, colors, or layout. No build step is required.
